Aplicação para venda de produtos web



a Neste software, o usuário poderá fazer vendas do seu comercio. Na venda seré relacionado um pedido, um cliente, o tipo de documento e itens a serem comprados.



Para uma maior completeza do sistema, também serão cadastrados: Produtos, Pais, Estado, Cidade, CEP, Grupo de Produtos, Tipo de Documento, Itens do Pedido e cliente.



Para o cadastro do endereço do cliente são usados os dados de país, estado e cidade. Para o cadastro de produto, usa-se uma referência de um grupo de produtos. Ao efetuar uma venda, o usuário deve relacionar o cliente que está efetuando a compra, quais os produtos e respectivas quantidades que estão sendo comprados e o tipo do documento (usado para classificar as vendas, ex: a vista, a prazo, cheque, crédito, débito).
